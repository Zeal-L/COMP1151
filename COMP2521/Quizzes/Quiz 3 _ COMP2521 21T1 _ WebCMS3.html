Height is measured in terms of the number of links in the longest path from the root of the subtree.

(a)

height(L) = 3, height(R) = 2

(b)

height(L) = 4, height(R) = 3

(c)

height(L) = 8, height(R) = 4

(d)

height(L) = 2, height(R) = 2

(e)

None of the above

Your response was correct.
Mark: 1.00

Many people got this wrong. You are supposed to count the links starting from the subtree root , not the overall root.

In this case, the longest path from node 10 to a leaf has 3 links, and the longest path from node 30 to a leaf has 2 links.
